ESPN评体育年度妙语 姚明吃炸鸡皇帝带天赋搬家 
　　新浪体育讯　2010年即将结束，在这一年中，体育界的风起云涌给我们留下了深刻的记忆。值年末，ESPN的Page 2专栏评选出了年度经典语录。下面就让我们来看一看体育界各路名人的经典发言吧。
　　20. “我是被迫着离开那个房间的，要知道我根本都不想去其他屋看了。时间真的不够，我们很兴奋，情绪高涨。”
　　——蒂姆-特波谈到新秀集训时与时任丹佛野马队教练的约什-麦克丹尼尔斯的会面时说。
　　Page 2点评：他的大学时代在东南赛区度过真是可惜了，要在大十区多好。
　　19. “我不理解枫糖条是怎么有那么大的吸引力的。”
　　——球员戈尔登-泰特凌晨三点“误闯”甜甜圈店，西雅图海鹰队教练在被问到对此事的看法时这样回答。
　　Page 2点评：实际上有很多地方误闯起来，后果要比甜甜圈店严重得多了。
　　18. “那些照片是在一年半之前拍的，自那之后我一定已经成长不少了。”
　　——开拓者队中锋格雷格-奥登回应自己在网络上疯传的不雅照事件。
　　Page 2点评：这话说得没有一点反讽的意味在里面。
　　17. “这一天太令人伤心了。在这样一个满是困难的世界里，虽然这不能和医疗、以色列、恐怖主义或者教育问题相提并论，世界不会因此而结束，但它仍然让人伤感。”
　　——圣弗朗西斯科轰鸣队因支付不起去德州亚瑟港路费的21000美元，被收回了当赛季的西部冠军称号，美国篮球协会(ABA)总裁乔-纽曼在评价这件事的时候说。
　　Page 2点评：我们认为，现在有人可以控制事情并让其良性发展了。
　　16. “如果体型巨大是罪名，我会判他有罪；但如果是以故意伤害起诉，我不认为罪名成立。”
　　——爱德华王子岛法官约翰-道格拉斯这样评价青年冰球球员克里斯-道伊尔。
　　Page 2点评：如果我们要开播一档体育界的真人秀法庭节目，这就是个不错的素材。
　　15. “如果你向一名女性发送隐私部位的照片，就给大力了……那可是十诫里面的训条。”
　　——篮球名宿查尔斯-巴克利毒辣评价布伦特-法弗的手机照片事件。
　　Page 2点评：巴克利的嘴也该一起被挂在名人堂的墙上的。
　　14. “现在我们就像二战时的德军。船来了，他们来了！我们拿着望远镜，高喊：上帝啊敌军入侵了！”
　　——田纳西大学橄榄球主教练德里克-杜利如此描述球队的艰难状态。
　　Page 2点评：杜利的这个比喻让人摸不着头脑。不过尼克-塞班(去年冠军亚拉巴马大学主教练)也曾拿911和珍珠港事件来说事。
　　13. ”都说了当时我在脱毛。“
　　——印第安那人梅根-巴恩斯在被州警官问道在车祸发生前为什么不踩刹车的时候说。
　　Page 2点评：巴恩斯女士确实是在为了穿比基尼脱毛，但那时她还在一边开车。她的前夫不幸被碾在了轮下，要知道那还是在她去看望现任男友的路上发生的事。
　　12. ”我每时每分都会赞美你们！！！因为你们就是这么对我的！！！你们想要我从中得到教训？？？怎么得到？？！！我永远不会忘记那个场景的！！永远不会！！！不过还是感谢关心……“
　　——比尔队外接手史蒂维-约翰逊在对钢人队一战中，错失反超比分的达阵，事后他在微博上写道。
　　Page 2点评：我们就直说了吧。史蒂维，你说上帝在天堂俯视这人间的一切，战争，压迫，饥饿还有无数的惨剧？得了吧，就算上帝也看橄榄球联赛，他也不会看比尔队的。
　　11. ”我把自己所有的每一分钱都投入了进来，花了超过一百万美元接近两百万在这张专辑上。它的花费和布兰妮的专辑差不多，甚至还要多，因为我希望它的质量也有那么高……专辑里的歌曲会给流行乐坛带来重大影响的。“
　　——真人秀名人海蒂-蒙塔格谈到自己进军音乐界时这样说。
　　Page 2点评：蒙塔格的专辑首周销量不足1000张——这个数据来源还是很可信的——她花在专辑上面的钱刚好又与她和她糟糕的丈夫斯宾塞-普拉特欠下的税款大致相等。
　　10. ”你要敢乱碰我，我就把你逮捕。“
　　——加州居民约翰-泰纳在圣迭戈国际机场安检处这样对安检人员说。
　　Page 2点评：我们想知道查尔斯-巴克利如果遇到这种情况会怎么说。
　　9. ”任何拿着刀的人都不能靠近我的朋友埃克赛尔-罗斯一步。“
　　——在一场枪花乐队的私人演唱会上，穷街乐队前主唱塞巴斯蒂安-巴赫在阻止了一名可能的袭击者之后说。
　　Page 2点评：巴赫什么时候放弃音乐了，倒是可以考虑考虑安全局的工作。
　　8. ”你真是个混蛋，你怎么能把腿放在这么容易被压着的地方？“
　　——阿根廷足球巨星迭戈-马拉多纳对自己的座驾撞上的一名记者说。
　　Page 2点评：马拉多纳展现了他如同兰迪-摩斯一般的车技以及何塞-坎塞科那样的反应——换句话说，他是我们专栏的最爱。
　　7. ”我已经65岁了，人们从来不对我说那些……请给我来电，有胆写下那些乌七八糟的东西的人都该死。“
　　——有校友给他寄来一封低俗下流的邮件，于是德州农工大学运动主管比尔-拜恩在一条语音信息里这样说。
　　Page 2点评：拜恩后来为这条信息致歉了，”我挂断电话之后就后悔了。“作为他的朋友，我们其实是想知道他是否觉得这样的反击很棒。因为这实在很棒。
　　6. ”我犯过错，确定无疑，不过我所期待的是有替罪羊来替我登上首页背上骂名。我想念莱恩-基芬了。“
　　——田纳西大学篮球教练布鲁斯-珀尔说。
　　Page 2：珀尔的招生策略随便你怎么抨击，但这个人实在是很善于跟球迷们布道啊。
　　5. ”我在这里和你说话的每一秒，本该都是花在华府饼干屋的时间。“
　　——田纳西大学前锋韦恩-奇斯姆在和媒体对话的时候说。
　　Page 2点评：如果我们的年度语录决选采用NCAA的赛制，田纳西大学毫无疑问地会坐上头把交椅。
　　4. ”我对大学橄榄球了解多少？我本人好比爆米花，大学橄榄球和我是没什么生意关系的。“
　　——俄亥俄州大校长E-戈登-基说，这句话让他从原本的第24位直升到这里。
　　Page 2点评：我们不会去争论的……两边都不会。
　　3. ”我的人生就像一场风暴，一场该死的飓风。我来到一座城，那里废墟一片。破坏非常大，而在风暴终于结束后，就像早晨醒来睁开眼睛……发生了什么事？“
　　——前重量级拳王迈克-泰森
　　Page 2点评：亨利-沃兹沃斯-朗费罗。华尔特-惠特曼。迈克-泰森。(前二者都是著名诗人)
　　2. ”我将把我的天赋带去南方海滩。“
　　——勒布朗-詹姆斯，宣布他准备加盟迈阿密热火队。
　　Page 2点评：自我认识缺失是一种与生俱来的东西，它还不会轻易消失。最近就有一句话说”在网上看到某某词之前，我根本不知道它是什么意思来着。“
　　1. ”我坐在马桶上想着我的第三本书要怎么写。“
　　——前棒球明星何塞-坎塞科。(此君退役之后活跃在娱乐圈和体育界的各个角落，行为出位，不计形象。)
　　Page 2点评：我们最经典的语录来了。
　　还有一些大家熟悉的篮球相关人物的经典发言：
　　”大一是我人生中最美好的七八个月，我和我的队友们就是雪城大学的披头士。“——卡梅罗-安东尼
　　”他是一个年轻不成熟的男孩子，大麻吸得太多。“——森林狼总裁大卫-卡恩这样评价刚被交易来的迈克尔-比斯利
　　”(关于篮网的振兴)，想想整件事吧，如果你想帮某支球队重建成功，那一定不会是篮网队。“——路易维尔大学主教练里克-皮蒂诺
　　”我又没有死，我正喝着啤酒吃着炸鸡呢。你们在期待什么，葬礼吗？“——姚明
　　”很高兴我也把我的天赋带来南方海滩了，下一场打灰熊。“——保罗-皮尔斯率队战胜热火队之后说
　　”这可能是肯塔基大学篮球史上最重要的一天，我们同时出了状元秀在内的五个首轮新秀。“——肯塔基大学主帅约翰-卡里帕利
